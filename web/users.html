The following tutorial shows users how to convert a sequenced image stack saved as a set of numbered images in a directory to a quantized version that has 4 tissue types.

Graphical User Interface Version

Step 1:
Start the MIA application. You should be presented with the blank black canvas like the image below.


Step 2:
Locate the Image I/O pane in the right window. Double-click the resources/ directory. Then double click the test-images/ directory, and finally double-click the brainstack/ directory. The screenshots below show the starting resources/ directory and the ending brainstack/ directory.



Step 3:
Now locate your starting image in the brainstacks folder and double-click on it and then select Load Canvas. For this tutorial we selected mrbrain-8bit052.png. After clicking Load Canvas you should now see the MIA canvas has loaded the image.


Step 4:
Let's reduce some of the noise. To do this we are going to apply a blur of 3.0. To do this simply enter a value of 3.0 into the Blur pane and click the Apply button. The image below shows the image after applying the blur filter.

Step 5:
Next, we will quantize the image into 4 segmements using the Quantize filter. Locate the Quantize pane and enter a value of 4 for the number of Bins. Then click apply. The image below shows the image after being quantized.


Step 6:
Lastly, we save the edited image to a different directory. I just went back one directory to the resources/test-images/ directory. Notice, I also save the image with a new name that I type in the image textfield: brain-052-new.png. You can name the file whatever you want, however make sure that it ends with either .jpeg or .png. Then click the Save Canvas button. The Image I/O browser should now show the new file that you saved. The image below shows the newly saved file in the Image I/O browser.

You have successfully processed a CT scan using MIA.
