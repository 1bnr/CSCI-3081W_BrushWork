Tutorial – Converting and Segmenting a CT Scan Using MIA

The following tutorial shows users how to convert a sequenced image stack (saved as a set of numbered images inside a directory) to a quantized version that has 4 tissue types.

Graphical User Interface Version

Step 1:
Start the MIA application. You should be presented with the blank black canvas like the image below.


Step 2:
Locate the Image I/O pane in the right window. Double-click the resources/ directory. Then double click the test-images/ directory, and finally double-click the brainstack/ directory. The screenshots below show the starting resources/ directory and the ending brainstack/ directory.



Step 3:
Now locate your starting image in the brainstacks folder, double-click on it and then select Load Canvas. For this tutorial we selected mrbrain-8bit052.png. After clicking Load Canvas you should now see the MIA canvas has loaded the image.


Step 4:
Let's reduce some of the noise. To do this we are going to apply a blur of 3.0. To do this simply enter a value of 3.0 into the Blur pane and click the Apply button. The image below shows the image after applying the blur filter.

Step 5:
Next, we will quantize the image into 4 segmements using the Quantize filter. Locate the Quantize pane and enter a value of 4 for the number of Bins. Then click apply. The image below shows the image after being quantized.


Step 6:
Lastly, we save the edited image to a different directory. I just went back one directory to the resources/test-images/ directory. Notice, I also save the image with a new name that I type in the image textfield: brain-052-new.png. You can name the file whatever you want, however make sure that it ends with either .jpg or .png. Then click the Save Canvas button. The Image I/O browser should now show the new file that you saved. The image below shows the newly saved file in the Image I/O browser.

You have successfully processed a CT scan using MIA, these steps can be repeated for any other image in brainstack/.

Command Line Version


To start this tutorial you will want a sequenced image stack saved as a set of numbered images inside a directory. This tutorial will use the brainstack directory (located in resources/test-images) but can be applied to any image stack you may want to use. Follow these steps below to complete the task.


Step 1: Open Terminal
Open up a terminal on your computer and move into the project folder. This can be done with the cd command, when you are inside the project directory you are now ready to run the program.

Step 2: Understand using MIA command line mode
When running in command line mode we pass MIA multiple arguments. We first pass it the input files (in our case this will be a directory of images), a filter to use (potentially multiple filters), and the output file (in our case we will make an output directory to store all the created images).
The input files can be accessed with: resources/test-images/brainstack/mrbrain-8bit###.png
The output files will be stored in a directory like so: output/brain-final###.png
The filters that we will use:
  blur with amount 5.0, this will slightly blur the image (reduce noise)
  quantize with amount 4, sort the resulting image into 4 segments

Step 3: Creating the output
You will need to create an output directory, the terminal command below will create a new directory named "output" inside the project directory:
mkdir output

Step 4: Running the commands
In your terminal the command below can be used to run the filters on all images inside brainstack, the ### is used to grab all images in the directory with the name "mrbrain-8bit followed by 3 digits:
./build/bin/MIA resources/test-images/brainstack/mrbrain-8bit###.png –blur 5.0 -quantize 4 output/brain-final###.png

Step 5: Check out the results
After this command executes, there should be 99 new filtered images inside of the output directory. These images will have been applied a blur/quantize filter. Below are two images, the first is the file mrbrain-8bit058.png (no filtering), the second is brain-final058.png (blur/quantize filtering) hopefully your output looks just like this!




You have successfully processed a CT scan using MIA command line mode.
